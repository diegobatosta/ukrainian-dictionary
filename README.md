## Introduction

This project is the backend for a Ukrainian dictionary that has Russian and English translations. It is written in
Kotlin with Spring Boot. The frontend must communicate with the backend via a REST API.

## Versioning

Use [semantic versioning](https://semver.org/). The public API is the REST API defined in the `controller`
package.
